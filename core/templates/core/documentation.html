eLearning Project Documentation

A brief description of the main components and features of the project.

1. Data Models

Description of how your models (User, Course, Review, etc.) are structured and their purposes.

  • User: Extension of the standard user model, including fields is_student and is_teacher.
  • Course: Model for courses linked to instructors.
  • Review: Model for student reviews of courses.
  • Material: Educational materials associated with a course.
  • StatusUpdate: Model for user status updates.

2. Web Interface

List of main pages and their purposes:

  • / - Home page
  • /student/dashboard/ - Student dashboard showing enrolled courses.
  • /teacher/dashboard/ - Teacher dashboard showing created courses.
  • /course/<id>/chat/ - Chat page for a specific course.
