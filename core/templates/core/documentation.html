Документация проекта eLearning

Краткое описание основных компонентов и функций проекта.

1. Модели данных

Описание того, как устроены ваши модели (User, Course, Review и т.д.) и для чего они используются.

  • User: Расширение стандартной модели пользователя, включающей поля is_student и is_teacher.
  • Course: Модель для курсов, связанных с преподавателями.
  • Review: Модель для отзывов студентов о курсах.
  • Material: Учебные материалы, привязанные к курсу.
  • StatusUpdate: Модель для обновлений статуса пользователей.

2. Веб-интерфейс

Список основных страниц и их назначение:

  • / - Домашняя страница
  • /student/dashboard/ - Панель управления студента, отображающая записанные курсы.
  • /teacher/dashboard/ - Панель управления преподавателя, отображающая созданные курсы.
  • /course/<id>/chat/ - Страница чата для конкретного курса.
